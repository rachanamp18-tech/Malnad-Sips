 # Malnad Cafe Stories – Version 1.0

Malnad Cafe Stories is a responsive café website inspired by the beautiful Malnad region of Karnataka.
The website showcases Malnad coffee culture, traditional dishes, and seasonal specialties in a modern and simple design.

## Live Demo: 
https://malnad-sips.vercel.app/

 ## Features

* Responsive café website design
* Hero section with introduction
* About section describing Malnad inspiration
* Menu highlights with different categories
* Seasonal stories section
* Gallery of café dishes and drinks
* Contact section with address and details

## Technologies Used

* HTML5
* CSS3
* JavaScript

 ## Version 1.0

This is the first stage of the project focusing on the frontend design and layout of the website.

 ## Future Improvements

* Interactive menu (click category to view dishes)
* Backend integration
* Online ordering system
* AI-based recommendations
